# Khalil Land WiKi
欢迎大家一起协助完善服务器的WiKi

## 安装

1.将项目克隆到本地

2.进入对应目录

3.安装依赖:

```sh
npm i
```



## 使用

```sh
# start dev server
npm run docs:dev
# build for production
npm run docs:build
# preview production build in local
npm run docs:preview
# update vuepress and theme
npm run vp-update
```
