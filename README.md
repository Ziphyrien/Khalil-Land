# Khalil Land WiKi

欢迎大家一起协助完善服务器的WiKi

## 安装

1.将项目克隆到本地

2.进入对应目录

3.安装依赖:

```sh
bun install
```

## 使用

```sh
# start dev server
bun run docs:dev
# build for production
bun run docs:build
# preview production build in local
bun run docs:preview
# update vuepress and theme
bun run vp-update
```
